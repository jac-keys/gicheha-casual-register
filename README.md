# Gicheha Casual Registry

## Overview

**Gicheha Casual Registry** is an Android application designed to help manage and track casual workers within the Gicheha Farm system.
The application allows supervisors and workers to log in and manage worker records efficiently.

The system simplifies the process of registering casual workers, tracking their details, and improving coordination between supervisors and workers.

---

## Features

* Supervisor login system
* Worker login system
* Casual worker registration
* Worker record management
* Organized data structure for farm operations
* Simple and user-friendly interface

## Installation

1. Clone the repository

```
git clone https://github.com/jac-keys/gicheha-casual-register.git
```

2. Open the project in Android Studio

3. Allow Gradle to sync

4. Run the application on an Android device or emulator

---

## Usage

* Supervisors can log in to manage worker records.
* Workers can log in to access their system interface.
* The application organizes casual worker data for easy management within the farm.
